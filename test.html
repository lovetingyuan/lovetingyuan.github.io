  2. Tell the differences between a byte, a character, and a string. Tell the differences between Unicode, UTF-8, UTF-16, GB2312, and GB18030

    • Byte就是字节，是计算机二进制存储的计量单位，8比特算作一个字节，单个字节编码的话共可以表示256个字符 character就是字符，是有现实语言含义的可读性良好的字母符号，但计算机没法直接存储字符，只能按照一定的规范把字符分为不同的集合并按顺序进行编码 string就是字符串，是由一系列字符组成的文本，通常采用线性的数据结构存储
    • Unicode定义了世界范围内使用的一系列字符集，它为每种语言中的每个字符设定了统一并且唯一的二进制编码； 而UTF-8, UTF-16都是其具体的编码算法，UTF-8是变长字节的编码方式，兼容ASCII码，对于像希腊字母，阿拉伯字母等用两个字节，像中日韩等文字用三个字节，少数采用四个字节； UTF-16采用两个或四个字符，在常用字符编码上比UTF-8要多花费一些空间； GB2312, GB18030是专门为中文编码而制定的国家标准，前者扩展了ASCII码，增加了常用的汉字编码，后者继续增加了更多的汉字，符号和少数民族文字等。

  4. What is the difference between Factory and Builder design patterns? What is the difference between Adapter and Decorator design patterns? Give an example for each above pattern

    • 工厂模式通常是调用独立的方法，该方法返回一个具体的对象，如jsx中使用的h函数就是工厂模式，每次调用都会返回新的vnode
    • 生成器模式通常用来生成一个复杂的对象，该对象由不同的组件组成，生成器可以定义每个组件如何产生并且如何将它们组合成完整的对象
    • 适配器模式用来实现两个不相容的接口可以在一起工作，通过将其中一个接口包装成新的接口来实现。例如两个版本的API，参数不同，但是功能相同，就可以将它们包装成统一的调用方式。
    • 装饰器模式用来增强现有类的能力，但是仍然保持原有类的接口和用法。例如在某个类的某个方法被调用时需要上报一个数据，就可以用类似`@log`的方式进行装饰而无需更改原有类的实现。

  6. Explain the differences between mutable and immutable objects

    • 如果改变了一个对象的数据，但是对象的引用仍然保持不变，那么它就是可变对象； 如果一个对象的数据不可改变，否则只能生成新的对象，那就是不可变对象 在React中，一个渲染性能优化的手段就是组件数据采用不可变对象，这样react可以通过判断对象引用的变化采取必要的渲染 在函数式编程中，大量使用了不可变数据
